## Multimodal Search System - Vuemantics

A smart camera roll application that revolutionizes how users search through their personal media collections by leveraging AI to understand content ***semantically***

Run `just` to see all available commands grouped by category.

```zig
# Development
just dev-up [ARGS]       # Start dev environment (attached)
just dev-start [ARGS]    # Start dev environment (detached)
just dev-down [ARGS]     # Stop dev containers
just dev-stop            # Stop dev containers (no cleanup)
just dev-build [ARGS]    # Build dev containers
just dev-rebuild         # Rebuild dev containers (no cache)
just dev-logs [SERVICE]  # View dev container logs
just dev-ps              # List dev containers
just dev-shell [service] # Shell into dev container (default: backend)

# Production
just prod-up [ARGS]      # Start prod environment (attached)
just prod-start [ARGS]   # Start prod environment (detached)
just prod-down [ARGS]    # Stop prod containers
just prod-stop           # Stop prod containers (no cleanup)
just prod-build [ARGS]   # Build prod containers
just prod-rebuild        # Rebuild prod containers (no cache)
just prod-logs [SERVICE] # View prod container logs
just prod-ps             # List prod containers
just prod-shell [service]# Shell into prod container (default: backend)

# Local (no Docker)
just run-backend         # Run backend with uvicorn
just run-frontend        # Run frontend with pnpm
just sync                # Install backend dependencies
just sync-dev            # Install backend dependencies with all extras

# Linting & Formatting
just lint                # Run ruff linter (backend)
just ruff [ARGS]         # Run ruff linter with args
just ruff-fix            # Auto-fix and format backend
just ruff-format         # Format backend code
just biome [ARGS]        # Run biome linter (frontend)
just biome-fix           # Auto-fix frontend

# Type Checking
just mypy [ARGS]         # Run mypy on backend
just tsc [ARGS]          # Run TypeScript check on frontend
just typecheck           # Run mypy (alias)

# CI / Quality
just ci                  # Run lint and typecheck
just check               # Run ruff and mypy

# Setup
just setup               # Run setup script
just install-frontend    # Install frontend deps
just install-backend     # Install backend deps

# Utilities
just info                # Show project info
just clean               # Clean cache directories
just prune               # Clean Docker system
just nuke                # Delete ALL Docker data (requires confirmation)
```

## Architecture
- **Backend**: FastAPI with async support
- **Database**: PostgreSQL with pgvector extension
- **Cache**: Redis for performance optimization
- **AI Models** (100% local, $0 cost):
  - Qwen2.5-VL-7B for multimodal image/video analysis
  - bge-m3 for semantic embeddings (1024-dim, 72% MTEB score)
- **Frontend**: React + TypeScript + Vite + SCSS
